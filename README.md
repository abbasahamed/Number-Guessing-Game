# Number Guessing Game (C++)

This is a simple console-based number guessing game implemented with C++.
A random number between 1 and 100 is generated by the program, and the user has to guess the correct number.

For every guess by the user, the program will give a message indicating whether the guess was correct or not.
- "Too high" will be displayed if the guess by the user is greater than the actual number.
- "Too low" will be displayed if the guess by the user is smaller than the actual number.
- "Correct" will be displayed if the user guesses the actual number.

This game will continue until the actual number is guessed by the user.

## Features
- Random number generation
- Handling user input
- Conditional statements
- Looping mechanism for the game

## Technologies Used
- C++
- Standard Library for C++ (`iostream`, `cstdlib`, `ctime`)
